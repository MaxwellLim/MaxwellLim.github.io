Projects

Spoonified - Senior Design Project
This project included working with a team consisting of both computer science and electrical engineering majors on the design of a smart spoon which would track the pace and weight of bites that a person takes during a meal. I worked on the embedded code using C++, for the sensors of the spoon to be able to take the raw data and translate it to something which can be used to effectively track these metrics and communicate with a web app.
Rules Based Chatbot
The project task was to create a web scraper and rules based chatbot using nlp techniques. In this project I used python with many packages such as NLTK and BeautifulSoup. I created a web crawler and scraper using BeautifulSoup to extract the text and hyperlinks from the html code. I then processed the text using various nlp techniques and TF-IDF to extract the 40 most important terms from all of the texts that were scraped. With these terms, I created a small knowledge base with terms and sentences. For the chatbot, I made an if-else tree to create a dialog flow with regex to guide it. The chatbot would also store the users information such as their name and number of visits, as well as the answers to a short survey when they close the chatbot.
